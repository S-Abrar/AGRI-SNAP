# AGRI-SNAP
AgriSnap: An AI-powered mobile application for instant crop disease identification and management recommendations. Snap, diagnose, and save your harvest
AgriSnap is a comprehensive solution designed to empower farmers and agricultural enthusiasts. Using machine learning/deep learning (e.g., TensorFlow, PyTorch), the platform instantly analyzes uploaded crop images to diagnose common diseases and pests. It provides real-time management recommendations, severity scores, and a geolocated log of past diagnoses to help monitor and protect yields.
